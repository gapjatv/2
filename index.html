갑자TV 운영 개편 안내

시참 · 멤버십 · 룰렛 시스템이 아래처럼 바뀌어요.
현재 베타 기능이라 운영 상황에 따라 추가로 조정될 수 있어요.

01
선착순 채팅 게임
중단

방송 시작 시 첫 채팅 1·2·3위에게 진행하던 손풀기 감사 게임이에요.

아래 멤버십·시참권 개편으로 운영을 중단해요.
02
!멤버십 시참권
개편

등급별 시참권을 월 단위 개수가 아니라 일정 시간마다 1회 사용하는 방식으로 바꿔요.

☕ 갑자 라떼
변경 전
월 2장 지급 · 1일 1회 사용
변경 후
48시간(2일)마다 1회 시참
🍲 갑자 국밥
변경 전
월 4장 지급 · 1일 1회 사용
변경 후
24시간마다 1회 시참
명령어는 !멤버십 / !맴버십 등 오타로 입력해도 정상 인식되어 순번대기로 시참이 진행돼요.
03
시참권 사용
유지

!사용 또는 !시참권사용 명령어로 그대로 사용할 수 있어요.

멤버십 시참 다음 순서로 선착순 진행, 1일 1회 한도는 동일해요.
04
룰렛 시참
유지

멤버십 → 시참권 순서가 모두 끝나고 다음 순번이 없을 때, 룰렛으로 시참이 시작돼요.

05
!추첨 당첨 확률
조정

채팅에 !추첨 입력 시 시참권에 당첨될 확률이에요. 실패할 때마다 확률이 누적 상승해요.

변경 전
기본 1% 당첨 (방장·부방장 당첨 시 50% 확률 적용)
실패마다 0.1%p 증가 · 상한 없음
→
변경 후
기본 조건은 동일하고,
실패마다 0.05%p 증가 · 최대 5%로 상한
06
히든보스 처치 보상
조정

히든보스를 일정 수 이상 처치하면 시참권 1장을 지급하는 기능이에요.

변경 전
31마리 이상 처치 시 시참권 1장
→
변경 후
33마리 이상 처치 시 시참권 1장
07
시참권 입장 중 보스 처치 보상
삭제

시참권으로 들어와서 히든보스 33마리 이상을 처치해도, 더 이상 시참권을 추가로 지급하지 않아요.

고스펙 참여자가 시참권을 계속 재생산해 무제한으로 복제 사용하는 것을 막기 위한 변경이에요.

외국어 채팅 자동 번역 TTS

  • TTS 명단 지정 모드여도 항상 한글로 번역해서 읽어줌
09
한글→외국어 번역 채팅 (관제탑 전용)

백엔드 로그 위 입력창에 한글 입력 후 Enter 또는 전송 버튼.

  • 드롭다운에서 선택한 언어(영어/중국어/일본어/대만어 등)로 번역되어 접두어 없이 그대로 유튜브 채팅창에 전송됨
  • 전송과 동시에 번역된 문장을 해당 언어 음성 TTS로도 함께 읽어줌
10
숫자 코드 자동 처리
  • 채팅에 4자리 숫자만 입력 시 로그에 강조 표시
  • 버스 키워드(태초버스/신버스/지옥버스 등) 포함 시 유지
11
실시간시청자 (관제탑 전용)
  • 실시간 동시 시청자 수 + 채팅 참여자 목록 + 자체집계 탑팬 순위 확인
  • YouTube 정책상 시청만 하는 사람의 닉네임은 알 수 없어 채팅 작성자만 표시됨
  • 탑팬은 YouTube 정식 배지가 아닌 채팅/당첨/시참권/멤버십 기반 자체 집계임
